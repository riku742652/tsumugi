## Zaim CSV エクスポートフォーマット

Zaim プレミアムの「設定 → ファイル入出力 → ダウンロード」から取得できる CSV。
文字コード: UTF-8（BOMなし）、区切り: カンマ。

### カラム一覧（実CSVより確認済み・16列）

| # | カラム名 | 型 | 説明 |
|---|---------|-----|------|
| 1 | 日付 | YYYY-MM-DD | 取引日 |
| 2 | 方法 | 文字列 | **`payment` / `income` / `transfer`**（英語） |
| 3 | カテゴリ | 文字列 | 大カテゴリ（食費、交通費 など） |
| 4 | カテゴリの内訳 | 文字列 | 小カテゴリ（外食、電車 など） |
| 5 | 支払元 | 文字列 | 支出元の口座・財布名 |
| 6 | 入金先 | 文字列 | 収入・振替先の口座名 |
| 7 | 品目 | 文字列 | 商品名・品目（任意入力） |
| 8 | メモ | 文字列 | 自由記述メモ |
| 9 | お店 | 文字列 | 店舗名 |
| 10 | 通貨 | 文字列 | `JPY` など |
| 11 | 収入 | 数値 | 収入の場合の金額（0 if N/A） |
| 12 | 支出 | 数値 | 支出の場合の金額（0 if N/A） |
| 13 | 振替 | 数値 | 振替の場合の金額（0 if N/A） |
| 14 | 残高調整 | 数値 | 残高調整額（通常 0） |
| 15 | 通貨変換前の金額 | 数値 | 外貨取引の元金額 |
| 16 | 集計の設定 | 文字列 | **`集計に含めない`** の行はグラフから除外する |

### サンプル行（実データより）

```csv
日付,方法,カテゴリ,カテゴリの内訳,支払元,入金先,品目,メモ,お店,通貨,収入,支出,振替,残高調整,通貨変換前の金額,集計の設定
2026-02-12,payment,その他,未分類,"楽天銀行 カノン支店 普通 ****452",-,,,"ラクテンショウケンカブシキガイシャ",JPY,0,451000,0,0,451000,集計に含めない
2026-02-12,income,その他,-,-,"楽天銀行 カノン支店 普通 ****452",,,"ラクテンショウケンカブシキガイシャ",JPY,22000,0,0,0,22000,集計に含めない
```

### 重要な注意点

- `方法` は日本語ではなく英語: `payment` / `income` / `transfer`
- 数値カラムは空でなく `0` が入る（パース上は楽）
- `集計の設定` が `集計に含めない` の行は家計外取引（株買付など）→ **デフォルトで除外し、表示オプションで切り替え可能にする**

### 注意点

- `収入` / `支出` / `振替` の3カラムは排他的（1行に1つだけ値が入る）
- `計算対象` カラムが存在する場合がある（0=家計に含めない、1=含める）
- カテゴリが空の行（未分類）が存在しうる
- 同日に複数行存在する

## 技術選定候補

### フロントエンド

| 選択肢 | メリット | デメリット |
|--------|---------|----------|
| Vite + TypeScript | 型安全、開発体験良好、ビルド後は静的ファイル | Node.js 環境が必要 |
| 素の HTML + JS | ビルド不要、ファイルを開くだけで動く | 型なし、大規模化に弱い |

**推奨: Vite + TypeScript** — CSV の型定義が書けるため安全。

### CSV パース

| ライブラリ | 特徴 |
|----------|------|
| **PapaParse** | ブラウザ対応、日本語OK、型付き、シェア最大 |
| csv-parse | Node.js 向け、ブラウザ利用には工夫が要る |

**推奨: PapaParse**

### グラフ描画

| ライブラリ | 特徴 |
|----------|------|
| **Chart.js** | シンプルAPI、軽量、棒・折れ線・円すべて対応 |
| Recharts | React 専用、宣言的だが重い |
| D3.js | 高自由度だが学習コストが高い |

**推奨: Chart.js**

## 表示したいグラフ（案）

1. **月別 支出合計** — 棒グラフ（月ごとに支出の推移を把握）
2. **カテゴリ別 支出内訳** — ドーナツグラフ（選択月のカテゴリ比率）
3. **収入 vs 支出** — 積み上げ棒グラフ（月ごとの収支バランス）
4. **カテゴリ別 月次推移** — 折れ線グラフ（特定カテゴリのトレンド）

---

## 制約・注意点

- ファイルはブラウザ内のみで処理（サーバー送信なし）— プライバシー上必須
- Zaim プレミアム限定機能のため、CSV フォーマットが将来変わる可能性がある
- 大量データ（数千行）でも動作するよう、描画は非同期で行う
